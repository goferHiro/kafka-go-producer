# Kafka-Go-Producer

A minimalistic Kafka go producer for  local and cloud hosted kafka servers.

## Prerequisites

1. Kafka-Server 
2. Kafka-Consumer

##Config

1. Kafka server 
    1. Topic 
    2. Hosted - localhost/IaaS (Infrastructure as Service)
2. Go 

## Usage

```bash
git clone https://github.com/goferHiro/kafka-go-producer/
cd kafka-go-consumer
go run .
```
